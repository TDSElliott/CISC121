Part 1: Superhero Kitten Name Generator

Overheard in a park ...
      Child 1: "Let's play superheroes!"
      Child 2: "Let's play kittens"
      Child 3: "Let's play superhero kittens!"

For this part of the assignment, you will write a small program which "generates" the name for a superhero kitten based on the user's name.

The program will first prompt the user for their name, and then use the ASCII values of the characters in their name to map it to a superhero prefix and a kitten postfix that have been provided below as lists. Your program should then output the combined superhero-kitten name. For example, your output might be:

"Super Whiskers"

Below are the requirements for this part of the assignment.

Requirements

  1. Your program must implement a function, askForName() which has one parameter. The function will ask for either the user's first name, or last name, depending on the parameter's value, and return it.
  2. Your program must implement a function, calculateASCIITotal() which has one parameter. The function should calculate the total sum of the ASCII values of all the characters in the given parameter using a loop, and return that value.
  3. Your program must implement a main() function which:
    1. Asks the user for their first name, then last name
    2. Determines the total ASCII values for each name
    3. Maps the first name to a superhero prefix from a list (hint: use a mathematical operator to calculate an index)
    4. Maps the last name to a kitten postfix from a list
    5. Outputs the combined superhero-kitten name

Lists of Superhero and Kitten Names

Please use the following lists of prefixes and postfixes in your assignment. Feel free to customize them.

listOfSuperheroPrefixes = ['Super', 'Ultra', 'Marvel', 'Mega', 'Laser', 'Lightning', 'Thunder', 'Power', 'Iron', 'Captain']
listOfKittenPostfixes = ['Fuzz', 'Whiskers', 'Purr', 'Paw', 'Claw', 'Fang', 'Snuggle', 'Cuddle', 'Cute Overload']

Part 2: Find the Space Junk

This question was inspired a little by the website If the Moon Were Only 1 Pixel.

For this part of the assignment, you have been given a series of extremely basic digital images (i.e. in the fictional .bdi format) representing photographs of space. These images are monochromatic and their files simply contain a grid of 0s and 1s, each representing one pixel.

Create a program which locates small pieces of space junk within these images. A piece of space junk has been characterised as just a single pixel with the value of 1, surrounded by pixels with the value of 0. Surrounding pixels include pixels directly above and below, left and right pixels, as well as diagonals.

Below are the requirements for this part of the assignment.

Requirements

  1. Your program must implement a function readBasicDigitalImageFile() which has one parameter. The function should read in the contents of the .bdi file (which is really just a text file) and return a 2D list containing the image's pixel values.
  2. Your program must implement a function isPixelSpaceJunk() which has three parameters: a 2D list, and two indices. The function should:
    1. Check the pixel at the given indices for a value of 1
    2. If the pixel is 1, check all of the pixel's "neighbours" (all eight, including diagonals) for a value of 0. There is no space junk on the borders of the image
    The function should return whether or not the pixel is space junk (as a Boolean).
  3. Your program must implement a main2() function which:
    1. Reads in the contents of a .bdi file into a 2D list
    2. Loops through each pixel in the 2D list and tests for space junk
    3. Prints out each pair of indices for pixels that match the criteria for space junk

Please use the provided .bdi file to test you program.
